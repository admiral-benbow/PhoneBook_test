**Программа Телефонный Справочник**:

При запуске основного скрипта - main.py, переносит пользователя в интерфейс справочника, представленный в консоли.
Взаимодействие с консолью происходит при помощи ввода необходимых цифр-команд для управления программой

**Пользователь может**:
1) Добавить свой контакт - с полями "Имя", "Фамилия", "Отчество", "Организацию", "Рабочий" и "Личный" телефоны. 
2) Выводить контакты из справочника по одной записи
3) Через меню вывода контактов по одной записи пользователь также может редактировать отдельно выбранный контакт (все его параметры)
4) Выводить несколько контактов сразу. По умолчанию на одной странице будет выводиться 5 контактов, однако пользователь может сам задать это значение (_доп. функционал, вне тех. задания_)
5) Искать контакт в справочнике по одному параметру или по их комбинации (вплоть до включения сразу всех параметров)

Данные справочника хранятся в .txt файле
Скрипт не требует сторонних библиотек, использует только встроенные - os и re 
Основной функционал описан в классе PhoneBook, также для удобства хранения информации контакта был создан класс Contact
